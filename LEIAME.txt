Para a compilação dos binários basta digitar  "make" no terminal. Se só é nesces
sário compilar o shell, digite "make ep1sh" e digite "make ep1" para ter só o binário
dos escalonadores. 

Rodar "make clean" depois desses comandos ira remover os arquivos
.o criados no processo de compilação.

Para rodar o shell basta digitar "./ep1sh" no terminal, se precisar de root para
alguma ação feita no shell rode ele usando "sudo ./ep1sh". 

O escalonador deve ser executado rodando "./ep1 <arg1> <arg2> <arg3> <optionalArg4>", onde <arg1> é um número de 1 a 3 (inteiro) que representa o tipo de escalonador a ser usado (1 = Shortest Job First, 2 = Round Robin, 3 = Escalonamento com prioridade), <arg2> é o nome do arquivo que irá servir de input para o escalonador (o final .txt ou parecido é obrigatório), <arg3> é o nome do arquivo de output do escalonador (o final .txt ou parecido é obrigatório) e por fim <optionalArg4> é um parâmetro extra que se for usando com a letra "d", irá mostrar na stderr a chegada de um processo no sistema, informando a linha do trace daquele processo; uso da CPU por um processo, informando qual o processo que começou a usar a CPU e qual CPU ele está usando; libe
ração da CPU por um processo, informando qual o processo que deixou de usar a
CPU e qual CPU ele está liberando; finalização da execução do processo, informan
do a linha que será escrita no arquivo de saı́da e a quantidade de mudanças de contexto no momento em que elas acontecem.
